# 📖 Sobre

Este repositório reúne todo o conteúdo desenvolvido durante o curso de **.NET Aspire**, explorando a construção de aplicações distribuídas utilizando os recursos mais recentes do ecossistema .NET.

Durante o curso foram abordados conceitos como:

- Aplicações Distribuídas
- Docker Compose
- Service Discovery
- Orquestração de Serviços
- SQL Server
- Entity Framework Core
- Background Services
- Redis
- OpenTelemetry
- Observabilidade
- Health Checks
- Integration Tests
- Lifecycle Events

Além do código-fonte, este repositório também serve como material de consulta para futuros projetos utilizando .NET Aspire.
